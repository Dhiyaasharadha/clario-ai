# Clario AI 🚀

Clario AI is an automated content repurposing engine that transforms long-form videos into short, viral-ready clips using AI.

## 🔥 Problem
Long videos contain valuable insights, but most users consume short-form content. Finding and editing the best moments manually is time-consuming.

## 💡 Solution
Clario AI uses AI to:
- Convert video to transcript
- Identify high-impact “viral moments”
- Generate short clips
- Create hooks and captions
- Suggest a content posting schedule

## ⚙️ Features
- 🎯 AI-powered clip detection
- ✂️ Automatic short video generation
- 🧠 Viral hooks & captions
- 📅 7-day content calendar
- ⚡ Fast and simple UI

## 🛠️ Tech Used
- React + Vite
- Node.js
- Google Gemini API
